## <h2 id="curriculum">课程列表</h2>

机器学习理论需要非常扎实的数学基础，包括优化，概率，分析等等，这边我们只列出机器学习理论的课程，由简到难，不同的课程中间会有一些重复交叉的内容，希望大家坚持下来。

如果大家认为有什么数学是必备的，可以告知我们，我们会补充在这一专栏，帮助大家更好的学习。
